## 📋 Overview 📖
**GlobeSwipe** is a visually immersive **countries slider UI** that showcases **32 countries** in a modern reel-style card interface.  

✨ Focused on:
- 🎨 Clean UI/UX  
- ✨ Smooth animations  
- 🖼️ High-quality visuals  

---

## ✨ Features 🌟

- 🌎 Showcase of 32 countries  
- 🎴 Reel / card-based slider UI  
- 🔄 Smooth swipe & navigation controls  
- 🎨 Modern and professional design  
- 🖼️ High-resolution images  

---

## ⚠️ Limitations 🚧

- 📱 Mobile version not available  
- 💻 Optimized for desktop only  

👉 Reason:
- 🖼️ High-resolution images  
- 🎨 Heavy animations
