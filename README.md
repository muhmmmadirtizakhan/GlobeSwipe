## 🌍 GlobeSwipe – Countries Slider UI

GlobeSwipe is a modern reel-style card slider UI that visually showcases **32 countries** 🌐  
The project focuses on smooth animations ✨, a clean layout 🎨, and high-quality visuals 🖼️.

### 🚀 Features
- 🌎 Covers **32 countries**
- 🎴 Reel / card-based slider design
- 🔄 Smooth swipe and button navigation
- 💼 Modern and professional UI
- 🖼️ High-resolution country images

### ⚠️ Important Note
📱 **Mobile version is not available**

**Reason:**
- 🖼️ High-resolution images  
- 🎨 Heavy graphics and animations  

Due to these factors, mobile optimization may affect performance and visual quality.  
Therefore, this project is designed for **desktop use only** 🖥️.

### 💻 Recommended View
- 🖥️ Desktop / Laptop screens
- 🌐 Latest versions of Chrome, Edge, or Firefox

### 🛠️ Tech Stack
- 🔹 HTML
- 🔹 CSS
- 🔹 JavaScript

### 🎯 Project Purpose
This project is created for **UI/UX practice and visual presentation**,  
where smooth animations ✨ and high-quality visuals 🖼️ are the main focus.
